## Code Block Tooltips

Code blocks now show a "Copy" tooltip when you hover over the clipboard button.

## File Size Warnings

The CLI will error when users try to add files to their docs bigger than 5 Mb.

If you need to display large videos, we recommend using a third-party service like Loom then embedding the video player on Mintlify.

## OpenApi Bug Fixes

Fixes bug related to parsing `openapi` property in page metadata.

## Segment

We added [Segment](/analytics/set-up/segment) as an analytics integration.
