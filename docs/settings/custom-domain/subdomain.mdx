## DNS Configuration

Navigation to the DNS settings for your domain. Add a new DNS entry with the following values.

```
CNAME | docs | cname.vercel-dns.com
```

<img alt="DNS" src="https://mintlify-assets.b-cdn.net/mintlify-dns.png" style={{borderRadius: "0.5rem"}} />

Please [contact support](mailto:hi@mintlify.com) if you don't see the custom subdomain set up after the above configuration.
