---
title: "Koala"
---

Add the following to your `mint.json` file to send analytics to Koala.

<CodeGroup>

```json Analytics options in mint.json
"analytics": {
    "koala": {
        "projectId": "required"
    }
}
```

```json Example
"analytics": {
    "koala": {
        "projectId": "mintlify"
    }
}
```

</CodeGroup>
