---
title: "Amplitude"
---

Add the following to your `mint.json` file to send analytics to Amplitude.

<CodeGroup>

```json Analytics options in mint.json
"analytics": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json Example
"analytics": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```

</CodeGroup>
